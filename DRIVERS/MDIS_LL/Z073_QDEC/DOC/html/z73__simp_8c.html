Detailed Description

Simple example program for the Z73 driver.

Author:
Christian.Schuster@men.de
Date
2010/04/21 15:50:15
Revision
1.5

This tool has two modes:

  • interrupt based:
    the interrupts of the module are enabled and the tool will display the data everytime a signal is received and data is read
  • polling mode:
    the interrupts of the module are disabled and the tool requests and displays data from the tool once a second

Required: libraries: mdis_api

Preprocessor Switches:
Z73_POSCNT_24 (enable 24 Bit position)
